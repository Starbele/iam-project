# IAM Roles and Secure Access Automation
Automates Azure identity and access controls using Azure CLI and Bash.

## Scripts
- `iam_setup.sh` — Creates resource group, VNet, subnets, AD groups, users, and role assignments
- `iam_cleanup.sh` — Revokes roles and removes all created resources

## How to Run
```bash
chmod +x iam_setup.sh && ./iam_setup.sh
```
